# Red Panda Asset Workspace

This folder is the generation workspace for red panda coaching art.

## Output Rules
- Format: `webp`
- Size: `1024x1024`
- Background: isolated/transparent
- Naming: `panda_state-<state>_variant-<x>_v<y>.webp`

## Tiering
- `tier1`: highest-frequency coaching states for realtime overlays.
- `tier2`: lower-frequency states for expanded coverage.

## Prompt System
- Base prompt template: `prompts/base-template.txt`
- State matrix: `prompts/state-matrix.csv`
