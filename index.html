      Help

      How do I install the app?

      Open in Safari, tap Share, then choose Add to Home Screen.

      Why isn‚Äôt the tuner listening?

      Tap the mic button in the Tuner view and allow microphone access. If you deny permission, use the reference tones.

      How do I keep progress?

      Checklists are saved per session. For permanent notes, print or save a PDF from the Parent Zone.

      Can I use this offline?

      Yes. Once installed, all tools and songs work without internet.
