# Design System

## Typography
- Display: Fraunces variable for headings.
- Body: Nunito variable for UI and body copy.
- Type scale: `text-xs` â†’ `text-3xl` with responsive clamps.

## Color
- Primary: Copper (#D56A3F) for CTAs and progress.
- Secondary: Deep teal (#3F768C) for tool accents.
- Accent: Gold (#F0C45A) for highlights.
- Backgrounds: Warm paper (#F6F2EC) and elevated surfaces (#FDF9F4).

## Components
- Topbar: Brand mark + status chips + nav.
- Cards: Default surface, used for tools, insights, and lists.
- Buttons: Primary, secondary, ghost.
- Chips and pills: Small status indicators.
- Progress ring: Visual session completion cue.
- Toggle: Preferences and switches.
- Meter: Tuner cents indicator.
- Metric bar: Insight visualizations.

## Layout Rules
- Single column on mobile, multi-column grids on tablet/desktop.
- Sections separated by generous vertical spacing.
- Sticky header on desktop for quick navigation.

## Accessibility
- All buttons and links have visible focus states.
- Toggle targets meet 44px min size.
- Reduce motion and high contrast preferences supported.
