# Design Tokens

## Palette
- `--color-bg`: #F6F2EC
- `--color-bg-alt`: #EFE7DD
- `--color-surface`: #FFFFFF
- `--color-surface-elevated`: #FDF9F4
- `--color-ink`: #1F1A17
- `--color-muted`: #6B5F57
- `--color-primary`: #D56A3F
- `--color-secondary`: #3F768C
- `--color-accent`: #F0C45A

## Typography
- `--font-display`: Fraunces variable
- `--font-body`: Nunito variable
- `--text-xs` → `--text-3xl`: responsive clamp scale

## Spacing
- `--space-1` → `--space-16`: 0.25rem to 4rem scale

## Radius
- `--radius-sm` 10px
- `--radius-md` 18px
- `--radius-lg` 26px
- `--radius-xl` 32px

## Shadows
- `--shadow-1`: soft elevation
- `--shadow-2`: medium elevation
- `--shadow-3`: subtle outlines
