# User Flows

Last updated: 2026-02-15

## Flow A: Daily Session
1. Open app → Overview.
2. Tap "Start session" → timer starts.
3. Complete each step in Flow list.
4. Use tuner/metronome as needed.
5. Finish session → duration stored + summary updated.
6. Write reflection → saved locally.

## Flow B: Tuner Use
1. Open Studio Tools.
2. Tap "Start tuner".
3. Grant microphone permission.
4. Read note + cents offset → adjust pitch.
5. Stop tuner.

## Flow C: Metronome + Tap Tempo
1. Open Studio Tools.
2. Adjust tempo slider or tap tempo.
3. Start metronome → audio click runs.
4. Pause or stop.

## Flow D: Record a Clip
1. Open Studio Tools.
2. Tap "Start recording".
3. Play a short clip.
4. Tap "Stop recording".
5. Clip saved to local library.

## Flow E: Coach Check-In
1. Open Insights.
2. Adjust manual sliders for intonation, rhythm, focus.
3. Review next-step recommendations.

## Flow F: Parent Planning
1. Open Library.
2. Add weekly goal + reward.
3. Save coach note.

## Flow G: Data Export
1. Open Settings.
2. Tap "Export JSON" or "Weekly PDF".
3. File downloads with sessions + notes.

## Flow H: Onboarding
1. First load shows onboarding dialog.
2. Review the four-step walkthrough.
3. Optional: check microphone access.
4. Start session.

## Flow I: Mic Permission Helper
1. Attempt tuner or recorder.
2. If access is denied, mic guide opens.
3. Retry after enabling permissions.
